A 2-stage classification pipeline for a food-service chatbot: first detecting out-of-scope questions, then classifying in-scope ones into 16 culinary intents (reservations, nutrition info, recommendations, etc.) using TF-IDF and Logistic Regression on the CLINC150 dataset. NLP · Text Mining

Aspect-Based Sentiment Analysis on McDonald's Reviews

Built an ABSA pipeline that segments reviews into sentences, weak-labels sentiment with VADER, tags 8 aspects (service, food, speed, cleanliness, etc.) via rule-based matching, and classifies each aspect's sentiment with a Linear SVM on hybrid word + character TF-IDF features (SMOTE-balanced). Reached 96.6% weighted F1, with service and speed surfacing as the most-discussed and most polarizing aspects — deployed as an interactive Streamlit dashboard.

LinearSVCVADERAspect Tagging
